## Что реализовано ✅

### Инфраструктура
- ✅ Настроен React Native проект с Expo
- ✅ Настроен TypeScript
- ✅ Добавлены основные зависимости:
  - React Navigation для навигации
  - Axios для API запросов
  - React Query для работы с данными
  - Zustand для управления состоянием
  - React Hook Form для форм
  - Expo Location, Camera, Barcode Scanner

### Структура проекта
- ✅ Создана базовая структура папок:
  - `src/config/` - конфигурация API клиентов
  - `src/types/` - TypeScript типы
  - `src/store/` - Zustand хранилища
- ✅ Настроены API клиенты (sharedApi, cafeApi)
- ✅ Создан базовый authStore

### Типы данных
- ✅ Определены базовые типы:
  - User
  - Cafe
  - Booking
  - Wallet
  - Transaction
